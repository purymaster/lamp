Password Guide

Guides Password Secure-Level Check Program for managing customer's safe Password.

Password Secure Level

Password Secure Level
Secure Level Description
Unusable Password does not meet the requirements
Weak Password is not secure enough
Fair Password is fairly secure
Strong Password is very secure

Password Requirements

  • Password must be a combination of 6~15 (uppercase/lowercase) letters, numbers and symbols without any spaces.
  • At least 3 more combinations required.
  • Usable symbols.

!?$%&*()@#{[|]}^_~

Not allowed to use as a Password

  • Blank(space) cannot be allowed for Password.
  • No more than 4 same or consecutive letters and numbers for Password.
  • Your ID, telephone number, and registration number cannot be used as your Password.
  • Account Password cannot be the same as your sign-in password.
